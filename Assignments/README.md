# Assignments

This repository contains day-wise programming assignments, practice exercises, and small GUI projects built with Python and PyQt.

## Folder overview

- Day2_ Assignment: Python practice programs covering lists, loops, matrices, number checks, sorting, and basic statistics.
- Day3_Assignment: Placeholder folder for Day 3 work.
- Day4_Assignment: A simple calculator application built with PyQt6, including the UI file and launcher script.
- Day5_Assignment: Four small GUI-based projects for a guessing game, password generator, encryptor, and decryptor.

## Notes

The README files inside each folder give a short summary of the files they contain.
