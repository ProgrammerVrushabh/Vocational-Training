# Vocational Training Repository

A comprehensive collection of Python programming assignments, classwork, and practical projects completed during vocational training. This repository showcases foundational and intermediate Python concepts including algorithmic problem-solving, GUI development with PyQt6, and interactive applications.

## 📚 Project Contents

### **Assignments**

#### **Day 2 Assignment** 
28 Python programming problems covering fundamental concepts:
- Prime number identification
- Number manipulation and algorithms
- Data structure operations
- Conditional logic and loops
- Input/output handling

**Files:** `que1.py` through `que28.py`

#### **Day 4 Assignment - Calculator Application**
A fully-functional calculator GUI built with PyQt6:
- **Features:** Basic arithmetic operations with graphical interface
- **Files:**
  - `Calc.py` - Main calculator application with PyQt6 UI
  - `calculator.ui` - UI design file
  - `run_calculator.bat` - Windows batch script to launch the application
- **Dependencies:** PyQt6

#### **Day 5 Assignment - Number Guessing Game**
An interactive guessing game application with graphical interface:
- **Features:** Gameplay interface with PyQt6 GUI, random number generation
- **Files:**
  - `Guess_game.py` - Game implementation
  - `GAME.ui` - UI design file
- **Dependencies:** PyQt6

### **Classwork**

Daily classwork materials and practice problems:
- **Day 4 Classwork:** `negation_sentence.py` - Natural language processing exercise

## 🚀 Getting Started

### Prerequisites
- Python 3.x installed
- PyQt6 library (for GUI projects)

### Installation

1. Clone the repository:
```bash
git clone https://github.com/ProgrammerVrushabh/Vocational-Training.git
cd VT
```

2. Install dependencies:
```bash
pip install PyQt6
```

### Running Programs

**Python Assignment Problems:**
```bash
python Assignments/Day2_Assignment/que1.py
```

**Calculator Application:**
```bash
python Assignments/Day4_Assignment/Calc.py
```
Or on Windows:
```bash
cd Assignments/Day4_Assignment
run_calculator.bat
```

**Number Guessing Game:**
```bash
python Assignments/Day5_Assignment/Guess_game.py
```
