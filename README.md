# Vocational Training Repository

This repository contains my vocational training work across Python programming, classwork exercises, GUI applications, and a customer analytics portfolio project. It shows both learning progress and practical deliverables in one place.

## Repository Highlights

- Python problem-solving exercises in `Assignments/`
- Classwork practice files in `CLASSWORK/`
- A retail customer behavior analysis project in `Project/`
- A GitHub-ready structure for sharing assignments, notebooks, SQL, dashboards, reports, and presentations

## Folder Overview

### Assignments

- `Day2_ Assignment/` - 28 Python practice problems from `que1.py` to `que28.py`
- `Day3_Assignment/` - string manipulation exercises
- `Day4_Assignment/` - calculator GUI project with `.py`, `.ui`, and batch launcher files
- `Day5_Assignment/` - PyQt6 game and utility projects for guessing, password generation, encryption, and decryption
- `Day6_Assignment/` - unit conversion program
- `Day7_Assignment/` - traffic light GUI project
- `Day9_Assignment/` - HTML chessboard project

### Classwork

- Daily practice files covering basic Python, string handling, simple automation, and UI exercises

### Project

- `Project/` - customer shopping behavior analytics project with notebook, SQL scripts, dashboard, report, and presentation assets

## Technologies Used

- Python 3.x
- PyQt6
- SQL
- Jupyter Notebook
- Power BI
- HTML

## Getting Started

1. Clone the repository:

```bash
git clone https://github.com/ProgrammerVrushabh/Vocational-Training.git
cd VT
```

2. Install any required Python dependencies for the file you want to run.

3. Open the relevant assignment or project folder and run the matching script, notebook, or desktop file.

## Project Usage

To explore the customer analytics project, open `Project/README.md` and follow the workflow there. The project folder contains the complete analysis package, including:

- `Customer_Shopping_Behavior_Analysis.ipynb`
- `customer_behavior_sql_queries.sql`
- `customer_behavior_dashboard.pbix`
- `Customer Shopping Behavior Analysis.pdf`
- `Customer-Shopping-Behavior-Analysis.pptx`
